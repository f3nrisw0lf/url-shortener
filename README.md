# SHORTEN

A simple web-based application that shortens and optimizes the Uniform Resource Locator's length for easier recall and tracing.

# Setup

1. Install **Node JS**

   ```rest
    https://nodejs.org/en/download/
   ```

2. Install Dependencies

   ```bash
    $ npm install
   ```

3. Start

   ```bash
    $ npm start
   ```
